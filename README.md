# AFH Finance — Loan Calculator

A self-contained 3-funnel loan comparison calculator (Car & Asset, Personal, Business) for AFH Finance.

- All lender data, logos, and matching logic are embedded in `index.html` — no build step, no dependencies.
- Works offline; just open `index.html` in a browser.
- Designed to be hosted on GitHub Pages and embedded into the AFH Finance website via iframe.

## Live site
Once GitHub Pages is enabled (Settings → Pages → Source: main branch), this is served at:
`https://YOUR-USERNAME.github.io/REPO-NAME/`

## Updating
To change a lender rate or detail, edit `index.html` and commit — the live site updates automatically.
